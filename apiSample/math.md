# 数学系Sample

## 絶対値を表示

```lua
local b = math.abs(-15)
```

bは15になります。

- - -

## ランダムに数字を出す

```lua
math.random(10)
```
ランダムに１から１０までの整数を一つ表示します。

- - -

## Sin 

```lua
math.sin(x)
```

−１から１までの数字をx（ラジアン）を元に出します。

- - -

## Cos

```lua
math.cos(x)
```

−１から１までの数字をx（ラジアン）を元に出します。

- - -

## Tan

```lua
math.tan(x)
```

tanを出します。

- - -

## pi

```lua
math.pi
```
πを出します。
